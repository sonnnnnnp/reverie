### Requirements

- [Docker](https://www.docker.com)
- [Volta](https://volta.sh)

### Getting Started

#### Setup

アプリケーションの準備

```shell
make build
```

アプリケーションの起動

```shell
make up
```

アプリケーションの停止

```shell
make down
```

#### Help

その他のコマンドは以下を実行して参照！

```shell
make help
```
